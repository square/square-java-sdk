## Customer Group Info

Contains some brief information about a Customer Group with its identifier included.

### Structure

`CustomerGroupInfo`

### Fields

| Name | Type | Description |
|  --- | --- | --- |
| `Id` | `String` | The ID of the Customer Group. |
| `Name` | `String` | The name of the Customer Group. |

### Example (as JSON)

```json
{
  "id": "id0",
  "name": "name0"
}
```
